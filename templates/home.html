Profiles with Flask & MongoDb




Basic WebApp with Python Flask and MongoDb







Endpoint Examples





Display Home Page:

/

List All Users:

/get

Query by Username:

/get/<id> - eg. /get/jamesf

Query by Job Title:

/get/job/<job> - eg. /get/job/doc

Query by Name:

/get/name/<name> - eg. /get/name/james

Add Users:

/add/<id>/<name>/<surname>/<age>/<job>
